# word of CHOICE

An interactive experiment that mints words and stores them on the blockchain.

## Project Structure

```
wordofchoice/
├── frontend/          # Frontend code (HTML, CSS, JS)
├── contracts/         # Smart Contract code
├── .cursor/          # Cursor AI configuration
└── README.md
```

## Development

- Smart Contracts: Solidity
